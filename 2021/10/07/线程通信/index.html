线程通信

1.volatile实现线程通信

由于volatile变量的可见性，所以可以在需要通信的地方修改volatile变量，在需要接收通信的地方监听（循环检测）volatile变量

class CommunicationByVolatile {
    private static volatile boolean flag = false;

    public static void main(String[] args) throws InterruptedException {
        Thread t1 = new Thread(() -> {
            int num = 10;
            while (num > 0) {
                num--;
                System.out.println(Thread.currentThread().getName() + "====" + num);
                if (num == 5) {
                    flag = true;
                }
            }
        }, "线程一");

        Thread t2 = new Thread(() -> {
            while (true) {
                if (flag) {
                    System.out.println(Thread.currentThread().getName() + "运行");
                    flag = false;
                    break;
                }
            }
        }, "线程二");
        t2.start();
        Thread.sleep(1000);
        t1.start();
    }
}

2.wait、notify实现线程通信

使用wait()方法，让当前线程进入等待（阻塞）状态

使用notify()、notifyAll()方法唤醒线程。

notify()：随机唤醒一个线程

notifyAll()：唤醒所有线程

唤醒线程不代表线程获取了锁，而是给允许线程去尝试获取锁。

class CommunicationByNotify {
    public static void main(String[] args) throws InterruptedException {
        Object o = new Object();
        Thread t1 = new Thread(() -> {
            synchronized (o) {
                int num = 100;
                while (num > 0) {
                    num--;
                    System.out.println(Thread.currentThread().getName() + "====" + num);
                    if (num < 50) {
                        o.notify();
                    }
                }
            }
        }, "线程一");
        Thread t2 = new Thread(() -> {
            int num = 100;
            while (num > 0) {
                synchronized (o) {
                    if (num == 80) {
                        try {
                            o.wait();
                        } catch (InterruptedException e) {
                            e.printStackTrace();
                        }
                    }
                    num--;
                    System.out.println(Thread.currentThread().getName() + "====" + num);
                }
            }
        }, "线程二");
        t2.start();
        Thread.sleep(1000);
        t1.start();
    }
}

3.Condition实现线程通信

class CommunicationByLock {
    public static void main(String[] args) throws InterruptedException {
        ReentrantLock lock = new ReentrantLock();
        Condition condition = lock.newCondition();
        Thread t1 = new Thread(() -> {
            lock.lock();
            try {
                int num = 100;
                while (num > 0) {
                    num--;
                    System.out.println(Thread.currentThread().getName() + "====" + num);
                    if (num < 50) {
                        condition.signalAll();
                    }
                }
            } catch (Exception e) {
                e.printStackTrace();
            } finally {
                lock.unlock();
            }
        }, "线程一");
        Thread t2 = new Thread(() -> {
            lock.lock();
            try {
                int num = 100;
                while (num > 0) {
                    if (num == 80) {
                        try {
                            System.out.println("===============================");
                            condition.await();
                        } catch (InterruptedException e) {
                            e.printStackTrace();
                        }
                    }
                    num--;
                    System.out.println(Thread.currentThread().getName() + "====" + num);
                }
            } catch (Exception e) {
                e.printStackTrace();
            } finally {
                lock.unlock();
            }
        }, "线程二");
        t2.start();
        Thread.sleep(1000);
        t1.start();
    }
}

4.LockSupport实现线程通信

LockSupport是通过public final class Unsafe类，直接操作内存进行的

当调用park时，先尝试直接能否直接拿到“许可”，即_counter>0时，如果成功，则把_counter设置为0,并返回。

如果不成功，则构造一个ThreadBlockInVM，然后检查_counter是不是>0，如果是，则把_counter设置为0，unlock mutex并返回:

否则，再判断等待的时间，然后再调用pthread_cond_wait函数等待，如果等待返回，则把_counter设置为0，unlock mutex并返回：

当unpark时，则简单多了，直接设置_counter为1，再unlock mutext返回。如果_counter之前的值是0，则还要调用pthread_cond_signal唤醒在park中等待的线程：

class CommunicationByLockSupport {
    public static void main(String[] args) throws InterruptedException {
        Thread t2 = new Thread(() -> {
            int num = 100;
            while (num > 0) {
                if (num == 80) {
                    LockSupport.park();
                    System.out.println("===============================");
                }
                num--;
                System.out.println(Thread.currentThread().getName() + "====" + num);
            }
        }, "线程二");
        Thread t1 = new Thread(() -> {
            int num = 100;
            while (num > 0) {
                num--;
                System.out.println(Thread.currentThread().getName() + "====" + num);
                if (num < 50) {
                    LockSupport.unpark(t2);
                }
            }
        }, "线程一");
        t2.start();
        Thread.sleep(1000);
        t1.start();
    }
}

5.CountDownLatch实现线程通信

使用AbstractQueuedSynchronizer实现，是一个共享式锁。

CountDownLatch只有一个构造器public CountDownLatch(int count)，该构造器目的是设置AQS中的state。

CountDownLatch实际就是每次执行countDown()方法就将AQS中的state减一，当state为0后，线程就从等待队列中出来。

class CommunicationByCountDownLatch {
    public static void main(String[] args) throws InterruptedException {
        CountDownLatch countDownLatch = new CountDownLatch(3);
        CountDownLatch countDownLatch1 = new CountDownLatch(3);
        Thread t1 = new Thread(() -> {
            int num = 100;
            while (num > 0) {
                num--;
                System.out.println(Thread.currentThread().getName() + "====" + num);
                if (num % 10 == 0) {
                    countDownLatch.countDown();
                    long count = countDownLatch.getCount();
                    if (count == 0) {
                        try {
                            countDownLatch1.await();
                        } catch (InterruptedException e) {
                            e.printStackTrace();
                        }
                    }
                }
            }
        }, "线程一");
        Thread t2 = new Thread(() -> {
            int num = 100;
            while (num > 0) {
                if (num == 80) {
                    try {
                        countDownLatch.await();
                    } catch (InterruptedException e) {
                        e.printStackTrace();
                    }
                }
                num--;
                System.out.println(Thread.currentThread().getName() + "====" + num);
            }
        }, "线程二");
        t2.start();
        Thread.sleep(1000);
        t1.start();
    }
}
